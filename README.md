# Babel Webpack Starter

A starter pack to build ES6 applications. It uses webpack, Babel and webpack-dev-server to compile and serve ES6 apps in the browser

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
